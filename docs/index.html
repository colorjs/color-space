API

One object, every pair wired: space[from][to](…values) — plain numbers in each space's conventional ranges, a plain array back. No classes, no parsing, no state.

import space from 'color-space'

space.rgb.oklch(255, 128, 0)      // → [0.732, 0.186, 53]
space.oklch.rgb(0.65, 0.25, 180)  // any of 151 × 150 pairs

space.oklch.name    // 'oklch'
space.oklch.range   // [[0,1], [0,0.4], [0,360]]

// one space only — tree-shaken, ~2 kB
import oklch from 'color-space/oklch.js'
oklch.rgb(0.65, 0.25, 180)
// per-space metadata, generated from the sources
import meta from 'color-space/meta.js'

meta.oklch.channels    // [{symbol:'L', min:0, max:1, name:'Lightness'}, …]
meta.oklch.refs        // defining spec / paper links
meta.oklch.illuminant  // 'D65' · .observer .referred .dynamic
// whole buffers in WASM — same API, pinned bit-for-bit
import space, { alloc } from 'color-space/wasm'

const buf = alloc(n)     // Float64Array(n*3) in WASM memory
space.rgb.oklch(buf)     // in place, zero-copy — 27 spaces
// every pair as a GLSL shader chunk
import { glsl } from 'color-space/gl'

const src = glsl('rgb', 'oklch')
// → self-contained: vec3 rgb_oklch(vec3 c)
//   paste into any fragment shader, no uniforms

Recipes

// to a CSS string — oklch / lab / hsl need no massaging
const [l, c, h] = space.rgb.oklch(255, 128, 0)   // [0.73, 0.19, 53]
el.style.color = `oklch(${l} ${c} ${h})`
// an even gradient: step through OKLab, not sRGB
const A = space.rgb.oklab(224, 32, 32)
const B = space.rgb.oklab(32, 96, 224)
const stops = Array.from({length: 7}, (_, i) =>
  space.oklab.rgb(...A.map((v, k) => v + (B[k] - v) * i / 6)))
// does an OKLCH colour fit sRGB? linear-RGB must stay in [0, 1]
const lin = space.oklch.lrgb(0.65, 0.25, 180)
const inSRGB = lin.every(v => v >= 0 && v <= 1)   // false — too saturated
// on-set to delivery: S-Log3 footage → a Rec.2020 master
space.slog3.rec2020(0.41, 0.41, 0.41)   // → [0.408, 0.408, 0.408]

Skipped

A few famous systems would each need data or math no open formula supplies — so here is what's left out, and why.

Licensed swatch catalogs — Pantone®, NCS®, RAL Classic, HKS®, Toyo, British Standard — are defined by physical printed swatches and trademarked; their CIE measurements aren't openly redistributable, so there's nothing lawful to convert against (the official sites hold the authoritative values).

Atlas systems — Ostwald, DIN 6164 — are defined only by a printed sample book (a colour atlas); no formula to convert to or from them was ever published.

Non-invertible appearance models — Hunt, Nayatani 95, LLAB, ATD95 — publish a forward transform but no closed-form inverse, so they can't round-trip the way every space here does (RLAB, the one early model that inverts cleanly, ships).

Historical spaces — CIE RGB (1931), NTSC (1953), S-Log 1, REDLog, Panalog, ViperLog, RYB, ANLAB — are shipped, marked 🕰 in the catalog as working history.
