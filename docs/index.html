API

import space from 'color-space'

space.rgb.oklch(255, 128, 0)      // → [0.732, 0.186, 53]
space.oklch.rgb(0.65, 0.25, 180)  // any of 156 × 155 pairs

space.oklch.name    // 'oklch'
space.oklch.range   // [[0,1], [0,0.4], [0,360]]

// one space only — tree-shaken, 0.4-1.5 kB
import oklch from 'color-space/oklch.js'
oklch.rgb(0.65, 0.25, 180)
// per-space metadata, generated from the sources
import meta from 'color-space/meta.js'

meta.oklch.channels    // [{symbol:'L', min:0, max:1, name:'Lightness'}, …]
meta.oklch.refs        // defining spec / paper links
meta.oklch.illuminant  // 'D65' · .observer .referred .dynamic
// whole buffers in WASM — same API, pinned bit-for-bit
import space, { alloc } from 'color-space/wasm'

const buf = alloc(n)     // Float64Array(n*3) in WASM memory
space.rgb.oklch(buf)     // in place, zero-copy — 27 spaces
// every pair as a GLSL shader chunk
import { glsl } from 'color-space/gl'
import oklch from 'color-space/gl/oklch'

const src = glsl(oklch, 'rgb')
// → self-contained: vec3 oklch_rgb(vec3 c)
//   paste into any fragment shader, no uniforms
// any pair as a .cube LUT — Resolve, Premiere, FCP, OBS, ffmpeg
import { cube } from 'color-space/lut'

cube(space.slog3, space.rec709)   // 33³ — the header carries its measured accuracy
cube(space.rec709, space.rgb)     // pure transfer — automatically 1D, 4096 pt
// $ ffmpeg -i in.mov -vf lut3d=slog3-to-rec709.cube out.mov

The same color, 156 coordinate systems.

Web, print, film, broadcast, art, science, and history. Each entry carries conventional ranges, gamut shapes, provenance, known limits, and working code. Convert any space to any other: rgb to oklch, rgb to cmyk, slog3 to rec709, kelvin to rgb. No JavaScript required either — each dossier downloads its conversion as a .cube LUT that Resolve, Premiere, Final Cut, and OBS read.

  • Verified
  • 0 deps
  • Tree-shakeable
  • JS · GLSL · WASM · LUT
  • MIT

Not included

Licensed swatch catalogs — Pantone®, NCS®, RAL Classic, HKS®, Toyo, British Standard — are defined by physical printed swatches and trademarked; their CIE measurements aren't openly redistributable, so there's nothing lawful to convert against (the official sites hold the authoritative values).

Atlas systems — DIN 6164's constant-hue loci were only ever published as paywalled DIN data sheets, so nothing open defines it. (Ostwald ships: its ideal system — full colours as optimal semichromes — is pure mathematics over the CIE observer, no atlas needed.)
